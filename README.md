# Netflix Data Analysis

Проект для анализа данных Netflix с использованием Python и pandas.

## Описание

Анализ датасета с фильмами и сериалами Netflix:
- Распределение по типам контента
- Анализ по годам выпуска
- Анализ рейтингов
- Визуализация данных

## Технологии

- Python 3
- pandas
- matplotlib
- seaborn (опционально)

## Установка

```bash
pip install pandas matplotlib seaborn
```

## Использование

```bash
python netflix_data.py
```

## Данные

Датасет: `netflix_titles.csv`
